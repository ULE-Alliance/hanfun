9 ## Overview
10 
11 This a ULE Alliance's HAN-FUN Common Implementation public release.
12 
13 This software release allows third-party vendors to easily create devices that are compatible with the ULE Alliance HAN-FUN standard v1.3.0.
14 
15 Not all of the standard has been implemented so far, please see the *Features* section for further information on what has been implemented and what is missing.
16 
17 For more information on requirements, building and using this software package, please see the README file present in the software distribution.
18
